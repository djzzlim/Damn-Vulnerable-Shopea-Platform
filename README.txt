[ DESCRIPTION ]
Shopea is an intentionally vulnerable e-commerce application designed for 
beginners to practice web application penetration testing and Linux 
privilege escalation. 

[ OBJECTIVE ]
Your goal is to gain a foothold on the system and elevate your privileges
to root.

[ AUTHOR ]
Created by: Lim Kar Joon
Version: 1.0
Environment: Ubuntu 12.04 LTS (Legacy Lab)

[ VULNERABILITIES ]
- SQL Injection (Authentication Bypass)
- Unrestricted File Upload (RCE)
- IDOR (Insecure Direct Object Reference)
- Broken Authentication (Hardcoded Credentials)
